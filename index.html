Game Rules

The general set of rules I ended up utilizing was very simple. All the rolls were made with a single 20-sided die. None of the characters had any statistics that would affect the rolls, the only factor that could influence the final outcome besides the actual roll was the amount and quality of the player's description of their actions.

All the weapons usable by the PC's and NPC's had a set damage value associated with a succesful hit as well as a set HP value. The players each began with 30 hit points and the amount for various NPC's varied depending on both the NPC in question as well as my judgement as the GM. I would occasionally increase or lower the HP values on the fly depending on the situation going on.

The generic target number system was also quite simple, as the default number would always be 10. Depending on the difficulty of the action, and the reasonable likelihood of success, the target number would be increased or lowered accordingly on the fly by me (the GM). Roleplaying the situation well would always result in a lower target number for the player in question.

Combat was also very simplified, the system I came up with had the players roll for initiative, determining the turn order for the rest of the encounter. Every combat encounter was designed so that the players always got the first turn. After that the turns would take place one after another, and each character would be allowed one major action per turn. An attack, major movement or some other specified action would take up the entire turn, for players and enemies both.

In addition to these simple rules, each player also had a pool of 3 "hero points" available to them. These points were one-time-use only and would only replenish between sessions. The initial idea was that they would have to declare the usage of said points before making a roll, and it would guarantee success for that roll. This was given some leeway as players tended to forget that they were supposed to state it beforehand and I didn't want to be too punishing. This system was utilized much more during the second session once the players were a little more used to the mechanics and rules.

The main idea behind the various tools used for the sessions was to have as little bloat as possible. The fewer new systems players would have to learn to use, the more they could focus on what mattered. With this in mind, I ended up with the following:

Primary Tools for the Sessions

The two main platforms for communication were Zoom for voice comms and Discord for text chat. With this in mind I created a Discord server and invited all of the players there beforehand so everyone could read through the backstory and get used to the layout of the server.

Zoom was used as the primary voice communication platform, although as we had some minor technical issues we also used Discord temporarily for the same purpose as well.

As for the dice rolls, I had added a dice bot to the Discord server which took care of all our dice rolling needs via simple commands in text chat. This way everyone could see the results for both the PC's and NPC's.

The final tool I utilized to bring some added flavor and quality of life features to the second session was Owlbear.rodeo. It is a website that allows easy access to some basic DnD-style tools for both the GM and the players.

The main use for Owlbear was displaying rough overview maps (drawn by me) of the areas the players were in and covering undiscovered areas in "fog" so as to keep them hidden from the players until they proceeded further.

Backstory and Timeline

The Timeline for World 1:

2021 -> 2500: Major advances in genetics, science and technology leads to a future where countries are irrelevant, and companies are the major governing force. Gene splicing and advanced genetic enhancement has become the norm as more and more people rush to enhance themselves with animalistic traits and other non-human qualities.

2500-2599: People are starting to worry more and more about the long-term and hereditary issues modified genetics might cause and the market comes crashing down. As most major companies have banked everything on this being the way of the future they start working on the concept of supercities, nation-like cities that provide shelter and infrastructure to the people who are growing ever more restless due to the divide between genetically altered people and “normal” people. Society as has been known until this point starts to collapse and riots and brawls in the streets grow more common every day.

2600: The first supercity finishes construction. It is named The First City and people fight for the right to relocate into the city. Construction on additional cities proceeds along. A massive divide now separates enhanced and non-enhanced humans. Regular people now consider the altered sub-human and mass discrimination is commonplace. The small amount of altered people and their families that got into the First City before its gates were closed reside in slums at the edges of the city.

2800: A breakthrough in cybernetic technology allows enhancement to become commonplace. The technology is expensive and as such only the wealthiest can afford it. The rush to adopt this new technology resembles the mad dash for genetic enhancement centuries ago but only a minority of the populace can afford cybernetics. Two more supercities finish construction and are named The Second and Third City, respectively. These new supercities provide refuge to the massive number of genetically altered populace still locked out of The First City, as well as providing an alternative for regular folk to relocate to. Each city is governed by a group of extremely wealthy ex-corporates who make the major decisions regarding the cities. Two more supercities are still under construction.

2900: The Fourth City opens its gates for people. As more and more people are living within the first three cities and the remaining populace living in small communities outside the existing cities and construction sites as well as moving into the fourth city en masse, nature as a whole is starting to disappear entirely due to all the pollution and waste dumped outside the cities. The cities themselves are having some issues with law enforcement as the ruling elite do not care about the average person’s troubles. As an answer to this, the now cybernetically enhanced ruling elite (ex-corporates and wealthiest people within each city) officially forms the Regime, a governing body to rule the First City utilizing their own networks for communication. This is somewhat of a test run before their plan to expand their reign to each of the other cities. The Regime also establishes RAMP, Regime Authorized Military Police force to both act as military task force for themselves and to take care of law enforcement in general.

2940: The Fifth City never finishes its construction as intended since the workers went on a strike to oppose the Regime’s more and more oppressive take on governing the existing cities. Regime expands its control to now cover all the existing cities as well as whatever skeleton of a framework already exists for the Fifth City, now serving as the last bastion for people who either couldn’t get into any of the four preceding cities or have escaped them in order to get a fresh start. The Fifth City is a city in name only, as every day living there is a struggle.

302X: The current year. Several small resistance factions have sprung up in the last 80 or so years since the Regime’s official establishment of control over the cities. The RFA in particular has made several successful strikes against Regime control in the Second City and is considered an officially recognized terrorist organization by the Regime. All RFA members and affiliates are wanted criminals in all of the cities. You are located in the Third City at this time as you receive a mysterious invitation from a contact claiming to belong to a small-time resistance group simply calling itself the Third City Resistance. Having done successful mercenary work in the past you are allured by the promise of a hefty reward and decide to head into the bar mentioned in the invitation.

Summary of the Play Sessions

Session 1:

The party gathers at a bar called The Dive, one by one, having received a letter from someone called the Stranger. As they get settled and exchange greetings, they are passed by a hooded figure telling them to follow. The party decides to follow this person into the adjacent alleyway. After some initial hesitation, the party reaches a small clearing further down the alley and presented with a rolled-open cloth holding various armaments. Each party member picks some weaponry (baton and handgun for Dad, handgun and knife for Rex, knife and SMG for Ava and a baseball bat and sawed-off shotgun for Jag’er). Immediately after the party is attacked by a group of three thugs armed with various melee weapons and combat begins. Some time and struggling later the assailants are all taken care of, and the party hears slow clapping from the furthest end of the alley. The hooded person they were initially following approaches, sporting a mask that covers any and all distinguishable features about themselves.

"Well done people, well done! Terribly sorry about this crude introduction but I had to make sure you were up to the job. Each of you have showed promise in the past but what I am offering truly requires you to get your hands dirty. I even went to great lengths to provide you with armaments each of you might prefer, and you of course understand why all of these must be easily concealable. RAMP activity in the Third City is through the roof these days and walking around with a weapon in hand is not the best idea.

Now onto my offering. The Third City Resistance has recently acquired some information about a large shipment of RAMP weaponry arriving in the city via the eastern gate. Normally this kind of a delivery would be made using an aircraft to prevent any trouble along the way but our associates at the Fourth City made sure this one would be carried by truck. The shipment will arrive in an unremarkable warehouse at the edge of the city where it will then be loaded into a cargo chopper and delivered to RAMP HQ that way. What I want you guys to do is prepare an infiltration into that warehouse as the cargo sits there overnight, take out the guards stationed there and give me the all-clear so our troops can secure the shipment before more RAMP forces arrive at the scene."

"We are quite a way away from the warehouse, but we will have to go our separate ways. We cannot risk being seen in that area before you give us the all-clear. I've provided a car for you to use, or if you prefer, you can brave the streets but fair warning, this is not the friendliest neighborhood in the City. Then again, you've clearly proven more than capable of handling yourselves."

The Stranger offers each party member a radio.

"The guards change shifts at nightfall, you should start your infiltration shortly after. When you are done, contact me with this. Good luck!"

After some further inquiries regarding the exact prize for this particular job (the Stranger promises further information and all the money anyone would need), the party decides to proceed on foot. After traversing for a while, getting a little more acquainted with each other and the various reasons for everyone being here, the party runs into a clearly very panicked person running out of an alley. This man has reptilian eyes and a fat wallet in his hands. Begging for the party to help him, he tries to explain how he needs the money for his child. The party peeks into the alley and notices a rather large man kneeling and grunting back there.

Jag’er decides to knock the thief out cold, take the wallet and proceeds to walk away. A giant of a man rushes out of the alleyway, asking about the wallet. The party quickly explains what happened and the man-giant performs a record-breaking sprint to catch up to Jag’er, who is by now 10 or so meters ahead of everyone else. The angry man grabs Jag’er with both of his arms and throws him back towards the rest of the party, who now enter combat. As the large man had some distance to cross, Dad quickly scanned his surroundings to find anything of use and decided in a moment of genius (or madness) to utilize some nearby bricks to build a wall between the party and the approaching man. Unfortunately, there were only so many bricks, so the end result was more akin to six or so bricks stacked on top of each other. With some confusion about Dad’s decision the large man keeps approaching the party, walking effortlessly through Dad’s construct. The party then proceeds to shoot the man dead, managing to do so just before he could land even a single blow.

As the party continues forward they discuss their next move. With the newfound funds Jag'er decides to offer a meal to everyone and the party soon spies a restaurant on the side of the road. As they enter Jag'er negotiates for some under the counter goods and, after some hesitation from the clerk, manages to establish trust by showing his feline legs, so as to guarantee that they are not in any way affiliated with RAMP. The clerk then presents various firearms to the group and Jag'er spends his money on a magnum revolver. Everyone has something to eat and recovers a little.

Some time after the group find themselves on an empty abandoned overpass, and as they are in the middle of crossing it, they hear a gunshot echo in the distance. Following the gunshot, a piece of concrete right next to them explodes to smithereens. One of them notices a laser pointer and deduces that there might be a sniper aiming right at them. The party takes cover and considers their surroundings, noticing several pipes and scaffoldings on the left side (right next to them) of the overpass. Rex takes the lead, jumping down to the initial piping and having confirmed that they indeed can support them the rest of the group follows. Everyone makes the three jumps required to land back on solid ground, some with more effort than others. The party then proceeds to carry on along an adjacent street, out of view from the sniper.

Some time later the group arrives at their destination. The target warehouse compound has a chainlink fence surrounding it with no noticeable holes on this side, and before exploring further the party decides to wait until nightfall as instructed by the Stranger. They approach a nearby abandoned store, with the idea that they could perhaps peek over the fence from the roof, and Ava attempts to climb up the fire escape. As she does, the fire escape crumbles and falls off, Ava doing her best to negate the majority of the impact. Dad being a medic attempts to heal her wounds, but all the stress so far makes him miss with his stitching needle and instead ends up dealing more damage to Ava. The party then decides to enter the abandoned building and rest until nightfall.

As night is upon them, they hear several vehicles approach the compound, the gate open and shortly after presumably the same vehicles leaving. They deduce that the shift has now changed and decide on a new approach for infiltration. The party splits into two, each following the fence in opposite directions and plan on meeting up at the other end. Each pair follows the fence all the way around, meeting in the middle again and notice a large enough hole to possibly crawl through. Jag’er takes the lead and goes in first but ends up getting stuck by his feline hips. As he struggles to get free, the party decides to give him a collective push to help him through and succeed, in the process ripping the hole just a tiny bit more open so the rest of the group can fit through without any issues. They proceed to do just that, and as they enter the compound, they notice a guard standing roughly five meters ahead of them, back turned towards the party.

End of session 1.

Session 2:

Besides the initial guard right in front of the party, they also notice six other guards standing in sight. Two guarding the east gate, two at the main gate and two by the entrance to the warehouse itself. The party decides on a stealthy approach, initially grabbing the guard in front of them into a stranglehold and then knocking them unconscious. Rex takes the outfit the guard was wearing and pretends to hold the rest of the party as prisoners. They approach the warehouse guards in this manner, and Rex silently whispers something to each of them, attempting to get them to follow the party inside. As they enter the warehouse, they notice a small office on the left side with a closed door and two more guards behind the truck that is standing in front of them. The party then decide on assassinating the two guards that are with them, stabbing both. One of the guards gets stabbed directly through the throat and crumbles to the ground without making a peep, but the other one appears to be wearing some kind of armor underneath his clothing. Dad then takes his electric baton and the lighting jolts through the guard’s body, killing them. The group quickly fit themselves inside the truck’s cabin, a tight squeeze for four people but they make it work and proceed to hightail it out of the compound. As they attempt to ram down the main gate, they initially crash into it and come to a stop, but the gate eventually gives in and the party gets away with the truck.

Some distance away from the warehouse, the party contacts Stranger and agrees on a nearby meeting place. While they wait for Stranger to arrive they congratulate each other on a job well done. They then hear approaching vehicles, but do not see who approaches since it is nighttime and there are no street lamps around. Stranger approaches the group and congratulates them. “Excellent work guys, everything went as smoothly as I had hoped. I believe it is now time for some well-deserved answers.” Having said this, the Stranger removes his mask. This reveals a face covered in severe cybernetic augments. “The Regime thanks you for your service, the Third City Resistance maybe not so much. You did just raid a warehouse and steal their weapons after all.” As this is happening the party realizes they are being surrounded by RAMP soldiers, at least eight assault rifles pointed directly at them. Jag’er tries to fight back, knocking one of the soldiers out with a punch that shatters his helmet. Jag’er then receives similar treatment, getting a rifle butt to the head and is knocked unconscious. The rest of the party suffers the same fate, and some time passes.

The group wakes up to the sound of people scurrying about. They see another group of armed people, this time sporting similar outfits to the guards at the warehouse. One person approaches them and explains that they are the real Third City Resistance (editor’s note: I kinda forgot to add it in-game but he was supposed to explain a little further how the weapons delivery was secured in the Fourth City by a collaborating resistance group and that’s why it was delivered by a truck, as RAMP forces use mainly aircraft for cargo delivery) and sarcastically remarks what a great job the party has done. He then offers a “choice”, meet up with the leader of this group, Laura Reeve, or be shot right here and now. The party agrees to meet up with Reeve and are assigned one TCR member to act as a guide.

As they are traveling towards the TCR safehouse where Reeve is located, the party notices a medical clinic advertising free treatment for everybody, including altered people. As they are a little worse for wear at the moment, the party decides to pop in. The TCR soldier guiding them stays outside to keep watch. The party goes inside, gets their wounds treated and as they are wrapping things up with the doctor a door opens behind the doctor and a small, skeletal child appears. The child whispers “Help me!” as the doctor kicks the door back closed again. Intrigued, the party questions the doctor about this. She explains that it’s nothing to worry about and since the party is all healed up now, they should leave. She then excuses herself and goes in the room the child appeared from. The party decides to follow, only to find the door locked. Ava ends up picking the lock open and the party goes in. There are children all over the room, all rather sickly-looking and clearly being experimented on by the doctor. Dad immediately confronts the doctor and asks what the hell is going on here. The doctor tries to explain how all of these kids are street orphans that nobody misses or wants, and that their sacrifice goes a long way towards the greater good. The party doesn’t buy it and confront the doctor further. She backs away and, in a panic, injects herself with a syringe she had on the table behind her. This visibly flies her into some kind of a frenzy, her eyes getting bloodshot, and she grabs a scalpel in each hand. Combat ensues. The doctor proved surprisingly resilient, most likely due to the injection she gave herself, but goes down eventually. As the party is about to leave Dad remarks to the children that they are free now and should stay here for a while. As the group meets up with their escort back outside, they inform him about the kids inside and he contacts someone on a radio. He then says: “Don’t worry, they will be taken care of by the TCR.” The party proceeds onwards.

As they reach the safehouse, they notice that it is an old apartment building converted into a makeshift base of operations, TCR members going about their business all over the place. They try to ask around where they could find Laura Reeve when she makes her presence known right behind them. Reeve explains that this one of the minor operating bases of the Third City Resistance. She also tells the party that the Stranger got away with the weapons, and that the Stranger is in fact the Officer, one of the people in charge of overseeing RAMP operations within the cities. He is also the person responsible for capturing Jag’er’s father, whom they still have in custody somewhere. She also explains how they had a tracker planted on the weapon crates so she can point the party in the direction of a small RAMP outpost where the weapons are temporarily being held. Seeing how they just lost all their new weaponry they were expecting they can’t offer better armaments to the party at this time, but they can provide a diversion if the party decides to assault the outpost directly. She gives the party a bomb that they should plant at the base after the weapons have been secured and can also provide a ride closer to the base.

Having taken up Reeve’s offer for the ride, the party is informed of what to look for. Apparently, the building in question has a seemingly unguarded front door. It is an undercover outpost, so they don’t want to draw any attention towards it. The building also has a garage entrance in the back, but there constantly two soldiers posted there. Lastly, there might be additional entryways on the roof of the building but that is TCR knows at this time. The party decides initially to focus on the possible rooftop entryways.

As they arrive at their final destination, the party notices that the information was accurate. There is a unguarded front door as well as an alleyway right beside the building with a fire escape just out reach. The group decide to collectively boost Jag’er up to the fire escape, and after some initial fumbling they succeed. Jag’er then takes a quick peek at the rooftop and notices a rooftop window as well as what looks like a ventilation shaft a little further. He helps the rest of the group up, and the party takes a look inside through the window. They see the truck and a large open garage space below them, but also notice four guards standing in said room. The group then decide to try their luck with the vent and manage to pry off the fan covering the vent. They crawl in, one by one, and pass by several gratings through which they can see some empty rooms as well as some guards. However, they have no context as to where these rooms are in relation to one another, so they carry on. The group reaches a dead end in the shaft and open a hatch in front of them. This leads them to a small bathroom.

From the bathroom they initially try to listen to any noise through the door, but nothing is heard in the immediate vicinity. They proceed through the door only to find themselves in a hallway, with several doors in sight. As they enter the hallway the party begins to hear gunfire and explosions some distance away, indicating the beginning of the distraction operation by the TCR. Now on a time limit, the party decides to explore the door on their immediate right first, carefully peeking in first. The room is dark, and the slight beam of light from the barely open door reveals a bed and someone lying on it. They then decide to turn on the lights in the room and realize that there are three soldiers sleeping there. The party quickly retreats back outside, turning off the lights in the process. The guards don’t appear to wake up. At this point the party remembers the bomb they were carrying with them and decide to plant it in the toilet. After this the party explores further, noticing a more heavy-duty door at the end of the hallway. The door is open, so they take look inside. This door leads to the garage area they saw from the roof. Instead of rushing in head-first, they decide to scout out the area a little further. Opposite from the garage door, they find a staircase leading up.

On the second floor they attempt to quietly open the door from the stairwell but end up somehow dropping the entire door from its hinges with an extremely loud bang. The party observes no immediate response. They then carry on, in another hallway. They find a door on their right and decide to go in. This door is opened somewhat sloppily as well, but luckily there appears to be no-one inside. The room is covered in various corpses however, and there is a lonely chair sitting in the middle of the room, with a spotlight trained at it. Blood spatters cover the floor around the chair as well as the chair itself. Some of the bodies are wearing TCR outfits. The party decides to fall back and see what the rest of the hallway has in store. They reach the end and find a door on the left side. Jag’er decides to kick open the door with force, but the door does not give in. They then try opening it normally and it is indeed open. The party goes in and find themselves in a somewhat large office room.

On the opposite side of the room they notice the Officer, in what looks to be a trance with several cables connecting from his arms and head to a computer in front of him. As the party prepares for combat the Officer comes to his senses and mocks the party for throwing away the small mercy he showed them by leaving them alive. He then removes his hooded jacket and reveals an arm cannon on his right arm and an arm blade on his left. After an initial exchange of fire, Rex makes an attempt at shooting the arm cannon. His weapon is not powerful enough however, and the bullet merely bounces off. Jag’er follows suit with his revolver and the shot manages to find its way directly into the barrel of the arm cannon. The entire arm explodes off, and the Officer is unable to use his cannon from here on. Instead, infuriated by the loss of his cannon he takes wild swings at the party with his arm blade, all the while the party is firing everything they’ve got at him. Everyone keeps consistently chipping away at the Officer, who shrugs off bullet after bullet, until Jag’er finally manages to blow a hole through the Officer’s right kneecap. The Officer retaliates with a special attack where he cleaves through the entire party not unlike a whirlwind. The party keeps returning fire, and eventually the Officer is bested by another shot from Jag’er going through his other kneecap. The Officer falls on his knees, barely conscious and babbles on about how this shouldn’t have happened.

As the party is in somewhat of a hurry at this point, they rush back downstairs to see what the rest of the building has in store. They come across an empty room initially and decide to each explore one room to make things go faster. Dad finds a kitchen and a clearly frightened chef cowering in the corner. Ava opens a door to the room closest to the front entrance, only to find two soldiers there looking at her in surprise. The soldiers notice how Ava is covered in blood, remember all the gunfire and noise from upstairs just now and quickly put two and two together. They decide to screw it all and run away, not willing to fight anyone who can beat the Officer. The next room contains a single guard playing video games in the corner, not noticing the party at all. The room also has a door to an armory of sorts, containing assault rifles and armor and such. The last remaining room has nothing but random junk inside, including chairs and wooden boards which the party uses to barricade the sleeping guards within their room.

Having exhausted all other options, the party enters the garage. They split up, two of them pouncing the guards by the garage doors and the rest taking care of the soldiers guarding the small side room. The side room is revealed to contain various gasoline barrels, gas cylinders and other explosive materials. The party decide to pour out a trail of gasoline to the garage door and do just that, after which everyone hops into the truck. As they begin to drive away, Dad flicks a match out of the truck’s window and the gasoline trail catches fire. The party drives away while the entire building explodes sky-high behind them.

Epilogue takes place in The Dive, the bar where everything began. Laura Reeve is there to congratulate the party on a job well done, even if they suffered one civilian casualty in the form of the chef. She also laments the loss of all the intelligence the Officer held as his remains, along with his computer and all the data was blown to bits. Otherwise everyone made it out more or less unscathed and the job got done.

End of session 2.

using System.Collections;
using System.Collections.Generic;
using System.Threading;
using UnityEngine;
using UnityEngine.SceneManagement;

public class PlayerScript : MonoBehaviour
{
    
    private Rigidbody2D rb;
    private bool facingRight = true;
    private float horizontalMove = 0f;

    
    [Header("Player Movement Settings")]
    [Range(0, 10f)] public float speed = 1f;
    [Range(0, 15f)] public float jumpForce = 8f;
    [Space]
    [Header("Ground Checker Settings")]
    public bool isGrounded = false;
    [Range(-5f, 5f)] public float checkGroundOffsetY = -1.9f;
    [Range(0, 5f)] public float checkGroundRadius = 0.3f;
	
	static Vector3 playerPosition;
	
	[SerializeField] private float duration;
	//[SerializeField] public GameObject hitBox;
	public PlayerAttack attack;

    void Start()
    {
        rb = GetComponent();
		Cursor.visible = false;
		Cursor.lockState = CursorLockMode.Locked;
    }

    void Update()
    {
        playerPosition = transform.position;
		
        if (isGrounded && Input.GetKeyDown(KeyCode.W))
        {
            rb.AddForce(transform.up * jumpForce, ForceMode2D.Impulse);
        }

        
        horizontalMove = Input.GetAxisRaw("Horizontal") * speed;

        
        if (horizontalMove < 0 && facingRight)
        {
            Flip();
        }
        else if (horizontalMove > 0 && !facingRight)
        {
            Flip();
        }
		
		if (Input.GetKeyDown("space"))
		{
			if (facingRight == true)
			{
				PlayerAttack attackHitBox;
				attackHitBox = Instantiate(attack, new Vector2(transform.position.x + 1f, transform.position.y), transform.rotation);
				Destroy(attackHitBox.gameObject, duration);
				
			}
			
			else if (facingRight == false)
			{
				PlayerAttack attackHitBox;
				attackHitBox = Instantiate(attack, new Vector2(transform.position.x - 1f, transform.position.y), transform.rotation);
				Destroy(attackHitBox.gameObject, duration);
			}
		}
		
		if (Input.GetKey("escape"))
		{
			Cursor.visible = true;
			Cursor.lockState = CursorLockMode.None;
			SceneManager.LoadScene("MainMenu");
		}
		
		
    }
    private void FixedUpdate()
    {
        
        Vector2 targetVelocity = new Vector2(horizontalMove * 10f, rb.velocity.y);
        rb.velocity = targetVelocity;

        CheckGround();
    }

    
    private void Flip()
    {
        
        facingRight = !facingRight;

        Vector3 theScale = transform.localScale;
        theScale.x *= -1;
        transform.localScale = theScale;
    }


    private void CheckGround()
    {
        
        Collider2D[] colliders = Physics2D.OverlapCircleAll
            (new Vector2(transform.position.x, transform.position.y + checkGroundOffsetY), checkGroundRadius);

        
        if (colliders.Length > 1)
        {
            isGrounded = true;
        }
        else
        {
            isGrounded = false;
        }
    }
	
}

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class NPCSpeech : MonoBehaviour
{
	
	[SerializeField] private bool playerIsIn;
	[SerializeField] GameObject dialogueBoxBackground;
	[SerializeField] private GameObject[] dialogueText;
	
	[SerializeField] private int dialogueIndex = 0;
	[SerializeField] int koko;
	
    // Start is called before the first frame update
    void Start()
    {
        
    }

    // Update is called once per frame
    void Update()
    {
		koko = dialogueText.Length;
		
        if (Input.GetKeyUp(KeyCode.F) && playerIsIn)
		{
			dialogueBoxBackground.SetActive(true);
			
			if (dialogueIndex >= dialogueText.Length)
			{
				dialogueText[dialogueText.Length - 1].SetActive(false);
				dialogueIndex = 0;
				NextDialogue();
			}
			
			else
			{
				NextDialogue();
			}
		}
		
		
		if (playerIsIn == false)
		{
			EndDialogue();
		}
    }
	
	void OnTriggerEnter2D(Collider2D collider)
	{
		if (collider.gameObject.CompareTag("Player"))
		{
			playerIsIn = true;
		}
	}
	
	void OnTriggerExit2D(Collider2D collider)
	{
		if (collider.gameObject.CompareTag("Player"))
		{
			playerIsIn = false;
		}
	}
	
	void NextDialogue()
	{
		if (dialogueIndex == dialogueText.Length)
		{
			dialogueText[dialogueIndex].SetActive(false);
			dialogueIndex = 0;
		}
		
		else 
		{
			if (dialogueIndex == 0)
			{
				dialogueText[dialogueIndex].SetActive(true);
				dialogueIndex++;
			}
			
			else
			{
				dialogueText[(dialogueIndex - 1)].SetActive(false);
				dialogueText[dialogueIndex].SetActive(true);
				dialogueIndex++;
			}
		}
	}
	
	void EndDialogue()
	{
		dialogueBoxBackground.SetActive(false);
		
		for (int i = 0; i <= dialogueText.Length - 1; i++)
		{
			dialogueText[i].SetActive(false);
		}
		
		dialogueIndex = 0;
	}
}

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class GuardianAI : MonoBehaviour
{
    public float speedOfWalking = 3;
    public int patrolDistance = 10;
    public Transform patrolledPosition;
    public bool movingRight;
	
	//Testing for HP stuff
	public int health = 3;
	
	//Experimental knockback from attacks
	private Rigidbody2D rb;
	public float knockbackForce;

    Transform player;
    public float chasingDistance = 10;

    /*used for debug*/
    //public float DistanceToPosition;
    /**/

    private bool patrolling = true;
    private bool chasing = false;
    private bool returning = false;
	
	//Used for flipping the sprite according to direction of movement
	private bool facingRight = true;

    void Start()
    {
        player = GameObject.FindGameObjectWithTag("Player").transform;
		rb = GetComponent();
    }

    void Update()
    {

        /*used for debug*/
        //DistanceToPosition = transform.position.x - patrolledPosition.position.x;
        /**/

        if (Vector2.Distance(transform.position, player.position) < chasingDistance)
        {

            patrolling = false;
            chasing = true;
            returning = false;
        }
        
        else if (Vector2.Distance(transform.position, patrolledPosition.position) < patrolDistance)
        {
            patrolling = true;
            chasing = false;
            returning = false;
        }
        
        else if (Vector2.Distance(transform.position, player.position) > chasingDistance)
        {
            
            patrolling = false;
            chasing = false;
            returning = true;
        }

        if (patrolling == true)
        {
            Patrolling();
        }
        else if (chasing == true)
        {
            Chasing();
        }
        else if (returning == true)
        {
            ReturnToPosition();
        }
		
		//Flips the sprite for the enemy while patrolling
		if (movingRight && facingRight)
        {
            Flip();
        }
        else if (!movingRight && !facingRight)
        {
            Flip();
        }
		
		//Checking to see if the sprite needs flipping while chasing player
		if (chasing == true && player.position.x > transform.position.x && facingRight)
		{
			Flip();
		}
		
		else if (chasing == true && player.position.x < transform.position.x && !facingRight)
		{
			Flip();
		}
		
		//If health drops to 0, destroy this enemy
		if (health <= 0)
		{
			Destroy(this.gameObject);
		}

    }

    void Patrolling()
    {
        

        if (transform.position.x > patrolledPosition.position.x + patrolDistance / 2)
        {
            movingRight = false;
        }
        else if (transform.position.x < patrolledPosition.position.x - patrolDistance / 2)
        {
            movingRight = true;
        }

        if (movingRight)
        {
            transform.position = new Vector2(transform.position.x + speedOfWalking * Time.deltaTime, transform.position.y);
        }
        else
        { 
            transform.position = new Vector2(transform.position.x - speedOfWalking * Time.deltaTime, transform.position.y);
        }
    }

    void Chasing()
    {
        transform.position = Vector2.MoveTowards(transform.position, player.position, speedOfWalking * Time.deltaTime);
    }
    
    void ReturnToPosition()
    { 
        transform.position = Vector2.MoveTowards(transform.position, patrolledPosition.position, speedOfWalking * Time.deltaTime);
        
    }
	
	//Once again this whole thing is used for flipping the sprite, courtesy of Juho's implementation in the player script.
	private void Flip()
    {
        
        facingRight = !facingRight;

        Vector3 theScale = transform.localScale;
        theScale.x *= -1;
        transform.localScale = theScale;
    }
	
	//Testing collider stuff out
	void OnTriggerEnter2D(Collider2D collider)
	{
		if (collider.gameObject.CompareTag("PlayerAttack"))
		{
			health = health - 1;
			rb.AddForce(transform.right * knockbackForce, ForceMode2D.Impulse);
		}
	}
}
